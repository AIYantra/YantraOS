# Introduction

Welcome to the **YantraOS** official documentation.

YantraOS is the world's first Level 3 Autonomous Agent Operating System built as a native orchestration layer on top of Arch Linux. It is designed to invert the traditional computing paradigm: instead of waiting for your input, the OS acts as an autonomous entity running a relentless **SENSE → REASON → ACT** cycle (The Kriya Loop).

This documentation covers the core architecture, security protocols, and integration guides necessary to understand and extend the YantraOS ecosystem.
